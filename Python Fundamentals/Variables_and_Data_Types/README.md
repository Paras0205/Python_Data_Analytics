# Variables and Data Types in Python

## Overview

Variables are used to store data in memory, while data types define the kind of value a variable can hold.

This module covers the fundamentals of creating variables and working with different data types in Python.

---

## Topics Covered

### 1. Variables

Variables are containers used to store values.

```python
name = "Paras"
age = 20
```

### 2. Data Types

Python supports several built-in data types:

#### Integer (`int`)

```python
num = 10
```

#### Float (`float`)

```python
price = 99.99
```

#### String (`str`)

```python
name = "Paras"
```

#### Boolean (`bool`)

```python
is_student = True
```

### 3. Checking Data Types

Use the `type()` function to check the data type of a variable.

```python
age = 20
print(type(age))
```

Output:

```python
<class 'int'>
```

### 4. Multiple Variable Assignment

```python
x, y, z = 10, 20, 30
```

### 5. Naming Rules for Variables

* Variable names can contain letters, numbers, and underscores.
* Variable names cannot start with a number.
* Variable names are case-sensitive.
* Avoid using Python keywords as variable names.

Valid Examples:

```python
student_name = "Paras"
age = 20
```

Invalid Examples:

```python
2name = "Paras"
class = "Python"
```
